Сравнение двух HTML-отчётов ScanOval: показывает, какие уязвимости устранены, а какие остались.  
Генерирует CSV‑файлы и HTML‑отчёт со сворачиваемыми таблицами.

Как использовать

1. Скачайте готовый `compare_scanoval.exe` из раздела [Releases](../../releases).
2. Запустите EXE, выберите отчёт ДО исправлений, затем ПОСЛЕ.
3. Результаты (CSV и HTML) появятся в той же папке.
